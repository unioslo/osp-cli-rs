# Release v1.4.5

## Summary

- First real public binary release of the Rust `osp` CLI.
- Aligns the Rust release line to `osprov-cli` version `1.4.5`.
- Ships the rebuilt UI and REPL surface with stricter release and coverage discipline.

## Highlights

- UI rendering is driven by explicit low-level knobs for chrome, table borders, help layout, and message layout.
- `expressive`, `compact`, and `austere` are now real presentation profiles rather than special-case behavior.
- The REPL now behaves consistently across invocation flags, aliases, help rewrites, cached plugin output, and shell-scoped history.
- Release tagging now requires synced version metadata, release notes, and changelog content.
